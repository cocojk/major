Issues to be discussed in this class:

1) What is the machine called computer?

- What is Computer Architecture, what is Computer Science and Engineering?

2) Computer performance evaluation and ISA design

3) RISC (MIPS) machine instructions (processor's external interface)

4) Processor internal design (high-level organization)

- Data path and control, pipelining, memory systems and cache memory

5) Short introduction to parallel revolution (multiprocessors, multicores, cloud)
